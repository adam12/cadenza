Arithmetic in Cadenza

For any argument that can use numbers you can always substitute arithmetic expressions in.

Example: inject statements

{{ 123 + 456 }} produces 579
{{ 1 + 2 * 3 - 4 / 5 }} produces 7
{{ (1 + 2) * ((3 - 4) / 5) }} produces -3
{{ "a" * 3 }} produces aaa
{{ alphabet.length - 1 }} produces 25
